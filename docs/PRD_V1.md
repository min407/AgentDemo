## 一、产品概述

### 1.1 产品定位

银行智能尽调Agent系统是一款基于AI大模型的企业风险评估工具，旨在自动化完成银行对企业客户的尽职调查工作，帮助客户经理快速生成专业的风险评估报告。

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| 效率提升 | 报告生成时间从2-3小时缩短至5-10分钟 |
| 分析标准化 | 6大维度AI分析，确保评估维度一致 |
| 成本降低 | 减少人工投入，降低培训成本 |

## 三、详细功能

### 3.1 企业信息查询

- **输入**：企业名称/统一社会信用代码/客户号
- **输出**：企业基本信息（名称、行业、注册资本、成立日期、法定代表人、经营范围、风险评级等）
- **数据源**：Mock数据（可扩展对接企业数据库）

### 3.2 交易流水分析

- **数据范围**：近3个月账户交易记录
- **分析维度**：
  - 账户余额
  - 收入/支出总额
  - 交易笔数
  - 大额交易明细

### 3.3 AI风险评估（核心功能）

基于通义千问大模型进行6维度风险分析：

| 维度 | 评估内容 | 评分范围 |
|------|----------|----------|
| 经营稳定性 | 成立年限、注册资本匹配度、经营周期 | 0-100 |
| 资金流向 | 资金流入来源、大额资金异常 | 0-100 |
| 行业风险 | 行业周期、政策影响、竞争格局 | 0-100 |
| 交易行为 | 交易频率、夜间交易、可疑特征 | 0-100 |
| 反洗钱筛查 | 可疑交易特征、客户身份识别 | 0-100 |
| 综合风险 | 综合以上各维度给出最终评级 | 0-100 |

**输出**：
- 风险等级（高/中/低）
- 综合风险评分（0-100分）
- 各维度评分及分析详情
- 红旗信号识别
- 关键发现
- 后续建议

### 3.4 报告生成与导出

- **报告内容**：
  - 企业基本信息
  - 风险评分进度条
  - 6维度评分卡片
  - 关键发现列表
  - 红旗信号（如有）
  - 结论与建议
  - 后续步骤
- **导出功能**：支持PDF格式导出

### 3.5 Agent流程可视化

实时展示4个处理步骤：
1. 查询企业信息
2. 分析交易数据
3. AI风险评估
4. 智能生成报告

## 五、接口设计

### 5.1 分析接口

**POST** `/api/analyze`

**请求**：
```json
{
  "keyword": "企业名称或统一社会信用代码"
}
```

**响应**：
```json
{
  "success": true,
  "steps": [...],
  "result": {
    "enterprise": {...},
    "transactions": {...},
    "risk_analysis": {
      "risk_level": "low/medium/high",
      "risk_score": 65,
      "dimensions": {...},
      "findings": [...],
      "recommendations": [...],
      "llm_analysis": {...}
    },
    "summary": {...}
  }
}
```

### 5.2 查询接口

**POST** `/api/query`

**请求**：
```json
{
  "keyword": "企业名称"
}
```
